Problem: Find the number of numbers that have even digits in an array

Solution: Algorithm Used

    I initialized count = 0, to store the count of number with even digits.
    I looped through each number in the array: I used .toString() to convert the numbers into a string. I used .length to get the length of the string which is the number of digits in a number. I used digitCount to store the number of digits in a number and checked if it is even. For even digits counted, count is incremented.
    I returned count to output the number of even digit numbers in the array.
